# 🎬 Anime Series Manager

A web-based application that allows users to create their own anime series collections and upload videos locally for personal viewing.

---

## 🚀 Features

* 🎥 Create custom anime series with your own titles
* 📂 Upload and manage videos locally
* 🧾 Organize episodes inside each series
* ⚡ Fast and simple UI for smooth navigation
* 💻 Works completely on local storage (no external hosting required)

---

## 🛠️ Tech Stack

* Frontend: React.js
* Styling: Tailwind CSS
* Backend: Node.js / Express (if applicable)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/your-username/your-repo-name.git
```

2. Navigate to the project folder:

```
cd your-repo-name
```

3. Install dependencies:

```
npm install
```

4. Start the development server:

```
npm run dev
```

---

## 📌 Usage

* Create a new anime series by entering a title
* Upload video files from your local system
* Play and manage episodes within the app

---

## ⚠️ Note

This project is designed for **personal use only**. Uploaded videos are stored locally and are not shared publicly.
